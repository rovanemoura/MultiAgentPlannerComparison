21.2.39 synonym-stream-symbol Function

Syntax:
synonym-stream-symbol synonym-stream    symbol

Arguments and Values:
synonym-stream - a synonym stream.

symbol - a symbol.

Description:
Returns the symbol whose symbol-value the synonym-stream is using.

See Also:
make-synonym-stream
